What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language that describes the presentation of HTML documents.

Without CSS

<h1><font color="red">Welcome</font></h1>

Welcome

With CSS

<h1 class="title">Welcome</h1>

.title {
  color: red;
  font-family: Arial;
}

Welcome

CSS Syntax

A CSS rule consists of a selector and a declaration block with property:value pairs.

selector {
  property: value;
  property: value;
}
h1 {
  color: blue;
  font-size: 24px;
}

Combinator & Attribute Selectors

Combinators and attribute selectors offer powerful ways to target elements based on their relationships or the presence of specific attributes.

Combinator Selectors

/* Descendant Selector */
div p {
  background: #e8f4fc;
}

/* Child Selector */
ul > li {
  font-weight: bold;
}

/* General Sibling Selector */
h3 ~ p {
  color: #555;
}

/* Adjacent Sibling Selector */
h3 + p {
  font-style: italic;
}

Attribute Selectors

/* Selects all inputs with a 'type' attribute */
input[type] {
  border: 2px solid #3498db;
}

/* Selects type="text" inputs */
input[type="text"] {
  background: #f0f8ff;
}

/* Selects links with href starting with 'https' */
a[href^="https"] {
  color: #28a745;
}

/* Selects elements with class containing 'btn' */
div[class*="btn"] {
  border: 1px solid red;
}
